# Noogler(?) 四個月心得

四個月過得好快。馬齒徒增？
競程隨筆四個字已經幾乎掛羊頭賣狗肉了，基本上沒什麼碰。
但我應該還不會想要改掉這個名字。這個網址應該也是，希望能維持著不動的狀態，也盡量不想考慮刪除文章或是關掉部落格。感覺是少數能在世界上留下痕跡的方法（？）

## 工作

談到工作，我覺得我還是相當不知道自己在做什麼。
被分配到的一個最主要的 project 是讓 AI 可以接上我們 team 的軟體。
原本的計畫是用 MCP，至少我還可以寫個 python 來描述一些事情。
但流行很快就變成 `SKILL.md` 了，目前端出來的東西感覺起來就是一份我們 team 的軟體的使用文件。
接上 LLM 的測試也覺得很迷茫，感覺自己掌控不了的因素太多，不理想的 case 一直很難看出到底是哪裡出問題或是應該怎麼改進，每次跑測試都要放在那邊等好久。
也不是什麼其他事都沒做，但感覺就好像做了雜事而已，domain knowledge 自從做完我的 first bug 之後好像就沒在增加了，總覺得一直和 team 其他人做的事有距離。（要增加 domain knowledge 也是提不起什麼勁，OS 跟計算機結構基本上是我最差的科目@@）
還有一個看起來很難的 project，感覺可能首先研究出 spec 就得花一個月，但這個 project 好像 priority 也很低，一年之後才打算要驗收，沒做出來好像替代品也很好用了。
目前的心態就是分派給我的工作也沒幾件（我也還沒有能力做什麼事），看起來也沒有一個有多硬的死線，適當的慢慢做就好了（躺平養老？）。

## 福利

換個話題來講。敝社有免費提供三餐，都是自助式的。早午晚餐中我最喜歡早餐，特別是每天都會有炒蛋（菜脯炒蛋/香菇炒蛋等等），很讚，每天報到。
另外早餐我都會倒一杯米漿或牛奶。不管健不健康反正喝起來很開心。
小小抱怨一下每次青菜裡面都有加一些薑或是香菇之類莫名奇妙的食材。
哈根達斯雖然不是每次去都有，但基本上可以拿到的頻率還算高。拿得到的口味不多就是了。努力克制不要吃太多（但還是有吃）

雖然現在已經看習慣了但仔細想想其實有不少工作用的設備福利。
例如每個人都有升降桌，另外申請可以有 32 吋的螢幕可以用，這個螢幕比想像中還有好多功能，可以幫筆電+手機供電之外還可以當作 hub，螢幕接上網路線之後透過跟充電同一條的 type-c 接線讓筆電可以用有線網路（網路一差做什麼都很痛苦，不太想用無線網路）。

WFH 也是一個好福利（座位畢竟是擠在一起的，雖然應該沒人要看我的螢幕但做事的時候都莫名的有偶包，例如用中文跟 LLM 問問題的畫面不太想切過去被看到之類的...）

之前一直抱怨 chromebook 很不習慣，最近趁亂（？）申請了一台 linux 的筆電。不知道為什麼申請的這幾台都有觸控螢幕的功能，感覺很多餘。有一個小缺點是得要多打很多次密碼。我的密碼是六個單字的 diceware，非常有自信很安全。每次在 swaylock 打密碼都感覺自己是 paranoid。

## misc

最近被說了好幾次瘦了或是壯，被稱讚是蠻開心的（都壓不住嘴角），應該是不知道為什麼放到 instagram 的那張七星山的照片拍得很好。
to be fair 跟大學比確實比較瘦。大學的時候每天都會好好吃三餐（？）但畢業吃自己的時期因為懶得出門加上睡覺作息很隨便，大概都只會吃一餐，不知道為什麼就瘦了一點。可以摸到以前摸不到的骨頭感覺很新鮮（？）知道這樣會瘦之後就開始有意識的吃少一點（但吃飯是人生享受，我不想禁止自己吃喜歡吃的東西，拉麵還是攝取攝取），目前暫時目標先是離開過重狀態。

最近近乎可以說是上癮的沈迷在音樂遊戲，每天去機廳報到。也許會沈迷在這上面的其中一個原因是音樂遊戲很容易有成就感（compare to 工作內容），畢竟這是設計拿來玩的遊戲。
雖然我一直抱怨我沒時間，但客觀的來看其實我已經算是有很多時間了。板橋不小但我的通勤就是板橋到板橋，坐個捷運半小時內一定會到，再加上我基本上秉持不加班的原則，所以自由時間（？）並不少。
（雖然我聲稱說要寫部落格或是寫其他 code 或是增長自己但感覺現在蠻接近墮落玩樂的）

工作之後才開始用信用卡。手機的 NFC 實在好方便。

現在最煩惱的問題大概是要睡好覺 -- 明明隔天要上班但卻很難在一點前睡著，被鬧鐘叫醒的工作效率也很差。
第二煩惱的問題大概是覺得自己長不大[^kako]。感覺我真的還是不擅於社交，心理上很難拓展交友圈。
