A. 106

Statement

зөҰдҪ $N$пјҢе•ҸдҪ жңүжІ’жңүжӯЈж•ҙж•ё$A,B$дҪҝеҫ—$3^A+5^B = N$пјҢ$N \leq 10^{18}$

еӣ зӮә$A,B$жңҖеӨҡйғҪжҳҜ$\log$йҮҸзҙҡзҡ„жүҖд»ҘдәӮжһҡиҲүе°ұеҘҪдәҶ
еҹәжң¬дёҠд№ҹдёҚеӨӘжңғжәўдҪҚ

B. Values

Statement

зөҰдҪ дёҖејөз„Ўеҗ‘ең–пјҢйӮ„жңүжҜҸеҖӢй»һдёҖй–Ӣе§ӢеҜ«зҡ„ж•ёеӯ—$a_i$
жҜҸж¬ЎеҸҜд»ҘжҠҠе…©еҖӢзӣёй„°зҡ„й»һ$x,y$дёҖеҖӢж•ёеӯ—+1дёҖеҖӢж•ёеӯ—-1
е•ҸдҪ жҳҜеҗҰиғҪи®“жңҖеҫҢз¬¬$i$еҖӢй»һеҜ«зҡ„ж•ёеӯ—жҳҜ$b_i$
$1 \leq N \leq 2 \times 10^5$
$0 \leq M \leq 2 \times 10^5$
$-10^9 \leq a_i, b_i \leq 10^9$

Solution

еҸӘиҰҒдёҖеҖӢйҖЈйҖҡеЎҠиЈЎ$a_i$зҡ„зёҪе’Ңе’Ң$b_i$зҡ„зёҪе’ҢзӣёеҗҢе°ұеҒҡзҡ„еҲ°
ж–јжҳҜз”Ё dsu з¶ӯиӯ·зёҪе’Ң

C. Solutions

дёҖй–Ӣе§ӢжғіиӘӘйҖҷйЎҢйЎҢж•ҳеҫҲй•·е…Ҳи·‘еҺ»еҒҡ pD
зөҗжһңжҳҜж°ҙйЎҢпјҢдёҚйҒҺйӮ„жҳҜWAж…ҳQQ

Statement

гҖҢзөҰдҪ $N$еҖӢз·ҡж®өпјҢи«ӢйҒёеҮәжңҖеӨҡжўқе…©е…©е®Ңе…ЁдёҚзӣёдәӨзҡ„з·ҡж®өгҖӮгҖӮгҖҚ
зҸҫеңЁжңүе…©зЁ®жј”з®—жі•$A,B$еҲҶеҲҘеҳ—и©Ұи§ЈжұәдёҠиҝ°е•ҸйЎҢпјҡ
- $A$жј”з®—жі•пјҡжҢүз…§еҸіз•Ңз”ұе°ҸеҲ°еӨ§жҺ’еәҸпјҢдёҰжҢүз…§й ҶеәҸиҖғж…®з·ҡж®өгҖӮеҰӮжһңзҸҫеңЁиҖғж…®зҡ„з·ҡж®өдёҚжңғе’Ңз•¶еүҚзҡ„и§Јзҡ„д»»дҪ•дёҖжўқзӣёдәӨпјҢеүҮе°Үе…¶еҠ е…Ҙз•¶еүҚзҡ„и§ЈдёӯгҖӮијёеҮәжңҖеҫҢзҡ„и§Јзҡ„еӨ§е°Ҹ
- $B$жј”з®—жі•пјҡжҢүз…§е·Ұз•Ңз”ұе°ҸеҲ°еӨ§жҺ’еәҸпјҢдёҰжҢүз…§й ҶеәҸиҖғж…®з·ҡж®өгҖӮеҰӮжһңзҸҫеңЁиҖғж…®зҡ„з·ҡж®өдёҚжңғе’Ңз•¶еүҚзҡ„и§Јзҡ„д»»дҪ•дёҖжўқзӣёдәӨпјҢеүҮе°Үе…¶еҠ е…Ҙз•¶еүҚзҡ„и§ЈдёӯгҖӮијёеҮәжңҖеҫҢзҡ„и§Јзҡ„еӨ§е°Ҹ

и«Ӣж§ӢйҖ дёҖзө„ијёеҮәдҪҝеҫ—гҖҢ$A$ијёеҮәзҡ„зӯ”жЎҲ - $B$ијёеҮәзҡ„зӯ”жЎҲ = $M$гҖҚ
жіЁж„ҸдҪ ж§ӢйҖ зҡ„з·ҡж®өз«Ҝй»һеҝ…й ҲжҳҜдёҚеӨ§ж–ј$10^9$зҡ„жӯЈж•ҙж•ёпјҢиҖҢдё”е…ЁйғЁзӣёз•°
$1 \leq N \leq 10^5$
$-N \leq M \leq N$

Solution

еӣ зӮә$A$жј”з®—жі•жңғжҳҜйҖҷеҖӢе•ҸйЎҢзҡ„жңҖдҪіи§ЈпјҢжүҖд»Ҙ$M<0$жҳҜз„Ўи§Ј
йҰ–е…Ҳ$B$жј”з®—жі•дёҖе®ҡиҮіе°‘жңғжӢҝдёҖеҖӢз·ҡж®өпјҢжүҖд»Ҙ$M=N$дёҖе®ҡз„Ўи§Ј
$M=N-1$зҡ„и©ұиЎЁзӨәжңүдёҖеҖӢжӢҝжүҖжңүз·ҡж®өзҡ„и§ЈпјҢд№ҹе°ұжҳҜжүҖжңүз·ҡж®өйғҪдёҚзӣёдәӨпјҢ$B$жңғе’Ң$A$жңүеҗҢжЁЈзҡ„ијёеҮәпјҢд№ҹжҳҜз„Ўи§Ј
е°Қж–ј$0 \leq M < N-1$зҡ„caseпјҢе…Ҳж§ӢйҖ $N-1$жўқе®Ңе…ЁдёҚзӣёдәӨзҡ„з·ҡж®өпјҢжҺҘи‘—иҖғж…®жҠҠеү©дёӢдҫҶзҡ„йӮЈжўқз•¶дҪңе·Ұз•ҢжңҖе°Ҹзҡ„пјҢи®“д»–е’ҢеҫҢйқў$N-1$еҖӢз·ҡж®өдёӯзҡ„еүҚ$M+1$жўқзӣёдәӨпјҢж–јжҳҜ$A,B$жј”з®—жі•зҡ„ијёеҮәе°Үжңғзӣёе·®$M$
жҲ‘еңЁиіҪдёӯдёҖй–Ӣе§ӢжІ’жіЁж„ҸеҲ°$N = 1$зҡ„caseпјҢзү№еҲӨ$N = 1, M = 0$жүҚAC QQ

D. Powers

Statement

зөҰдҪ й•·еәҰ$N$зҡ„еәҸеҲ—$A = (A_1,A_2,\dots,A_N)$д»ҘеҸҠ$K$
е°Қж–јжүҖжңү$1\leq X\leq K$
и«ӢијёеҮә$\left( \sum\limits _ {i=1}^{N-1}\sum\limits _ {j=i+1}^N (A_i + A_j) ^ X \right) \pmod{998244353}$

Solution

з…©иәҒжҺЁејҸеӯҗ
жҲ‘дёҚж“…й•·QQиҠұеҘҪд№…
йҰ–е…Ҳ$\sum\limits _ {i=1}^{N-1}\sum\limits _ {j=i+1}^N (A_i + A_j) ^ x = \frac{1}{2}\left(\sum\limits _ {i=1}^N\sum\limits _ {j=1}^N (A_i + A_j) ^ X - \sum\limits _ {i=1}^N (2A_i)^X \right)$пјҢе…ҲжҠҠ$i < j$зҡ„жўқд»¶жӢ”жҺү
з„¶еҫҢзҷјзҸҫ
$$
\sum _ {i=1}^N \sum _ {j=1}^N (A_i + A_j)^X = \sum _ {i=1}^N \sum _ {j=1}^N \sum _ {p=0}^X \binom{X}{p} A_i^p A_j^{X-p} = \sum _ {p=0}^X \binom{X}{p} \left( \sum _ {i=1}^N A_i^p \right) \left( \sum _ {i=1}^N A_i^{X-p} \right)
$$
е°Қж–ј$0\leq p\leq K$й җиҷ•зҗҶ$\sum\limits _ {i=1}^N A_i^p$е°ұиғҪ$K^2$з®—еҮәдҫҶдәҶпјҢзёҪжҷӮй–“иӨҮйӣңеәҰжҳҜ$\mathcal{O}(NK+K^2)$
AC CODE

еҫҢйқўйғҪжҳҜиіҪеҫҢжүҚеҜ«еҮәдҫҶзҡ„йЎҢзӣ®QQжҲ‘е°ұзҲӣ

E. Medals

жҲ‘иҰәеҫ—йҖҷйЎҢзңҹзҡ„еҫҲжңүи¶ЈпјҢжІ’еҜ«еҮәдҫҶеҘҪеҸҜжғң

Statement

жңү$N$еҖӢе“Ўе·ҘпјҢд»Ҙд»ҠеӨ©зӮәеҹәжә–пјҢз¬¬$i$еҖӢе“Ўе·Ҙжңғе…Ҳе·ҘдҪң$A_i$еӨ©пјҢжҺҘи‘—ж”ҫеҒҮ$A_i$еӨ©пјҢеҸҲе·ҘдҪң$A_i$еӨ©гҖҒж”ҫеҒҮ$A_i$еӨ©пјҢе‘ЁиҖҢеҫ©е§ӢгҖӮ
жҜҸеӨ©жңғй ’зҷјдёҖеҖӢзҚҺзүҢзөҰжңүдҫҶе·ҘдҪңзҡ„е…¶дёӯдёҖеҖӢе“Ўе·ҘгҖӮз•¶з„¶еҰӮжһңз•¶еӨ©жІ’жңүе“Ўе·ҘдҫҶе·ҘдҪңе°ұдёҚжңғй ’зҷјд»»дҪ•зҚҺзүҢгҖӮ
зҸҫеңЁжғіи«Ӣе•ҸдҪ пјҢеңЁжңҖдҪізҡ„жғ…жіҒдёӢпјҢжңҖе°‘иҰҒеӨҡе°‘еӨ©пјҢжүҚиғҪи®“жүҖжңүе“Ўе·ҘйғҪеҫ—еҲ°иҮіе°‘$K$йқўзҚҺзүҢпјҹ
$1 \leq N \leq 18$
$1 \leq K \leq 10^5$
$1 \leq A_i \leq 10^5$

Solution

жңҖиҝ‘жүҚvirеҲ°дёҖе ҙйңҖиҰҒHall’s theoremзҡ„йЎҢзӣ®XDйҰ¬дёҠе°ұз”ЁеҲ°
жҲ‘зҡ„и§ЈжҳҜиҖғж…®е°Қзӯ”жЎҲдәҢеҲҶжҗң
е°Қж–јдёҖеҖӢеӣәе®ҡзҡ„еӨ©ж•ё$D$пјҢжғіеғҸдёҖејөжңү$D+NK$еҖӢй Ӯй»һзҡ„дәҢеҲҶең–пјҢе…©йғЁд»ҪеҲҶеҲҘд»ЈиЎЁжҜҸдёҖеӨ©д»ҘеҸҠжҜҸеҖӢдәәзҡ„$K$еҖӢзҚҺзүҢпјҢдёҰдё”еҰӮжһңи©ІеӨ©еҸҜд»Ҙй ’зөҰйӮЈеҖӢдәәзҚҺзүҢе°ұйҖЈдёҖжўқйӮҠпјҢйӮЈйәјжҲ‘еҖ‘жғіиҰҒзҹҘйҒ“зҡ„е°ұжҳҜжҳҜеҗҰжңүдёҖеҖӢеӨ§е°Ҹ$NK$зҡ„еҢ№й…ҚгҖӮ
еҒҮиЁӯ$A$жҳҜ$NK$еҖӢй Ӯй»һдёӯзҡ„дёҖеҖӢsubsetпјҢ$\Gamma(A)$жҳҜе…¶й„°еұ…пјҢеҢ…еҗ«дёҖдәӣд»ЈиЎЁеӨ©ж•ёзҡ„й Ӯй»һ
йӮЈйәјпјҢж №ж“ҡHall’s theoremпјҢжңүйӮЈжЁЈзҡ„е®ҢзҫҺеҢ№й…ҚиӢҘдё”е”ҜиӢҘ$\forall A, |A| \leq |\Gamma(A)|$
дёҚеӨұдёҖиҲ¬жҖ§еҸӘйңҖиҰҒжһҡиҲү$2^N$зЁ®зө„еҗҲпјҢеӣ зӮәд»ЈиЎЁеҗҢдёҖеҖӢдәәзҡ„й Ӯй»һйҒёеҶҚеӨҡеҖӢйғҪдёҚжңғеҪұйҹҝй„°еұ…зҡ„йӣҶеҗҲ

жҺҘдёӢдҫҶе°ұжҳҜжҲ‘еӣ зӮәдёҚзҶҹиҖҢжІ’еңЁиіҪдёӯеҜ«е®Ңзҡ„йғЁд»ҪQQ
зӮәдәҶе°Қж–јжүҖжңүsubsetйғҪзўәиӘҚдёҚзӯүејҸзҡ„жўқд»¶пјҢжҲ‘еҖ‘еҝ…й ҲзҹҘйҒ“жңүеӨҡе°‘еӨ©жңғеҪұйҹҝеҲ°йҖҷеҖӢsubsetзҡ„й»һ
жіЁж„Ҹзӯ”жЎҲжңҖеӨҡжҳҜ$2K\sum A_i$пјҢеӣ жӯӨеҸҜд»Ҙй җиҷ•зҗҶжҜҸеӨ©еҸҜд»Ҙе°ҚжҮүеҲ°зҡ„й„°еұ…
еҒҮиЁӯ$Cnt_s$д»ЈиЎЁеңЁ$D$еӨ©е…§жңүеӨҡе°‘еӨ©е°ҚжҮүеҲ°$s$йҖҷеҖӢsubsetпјҢд»ҘеҸҠ$Day_s$д»ЈиЎЁжңүеӨҡе°‘еӨ©жңғеҪұйҹҝеҲ°$s$йҖҷеҖӢsubset
йӮЈйәј
$$
Day_s = \sum _ {t \& s \neq 0} Cnt_t
$$
з”ЁSOS DPжҲ–жҳҜиў«зЁұзӮәFast Zeta Transformзҡ„жҠҖе·§еҸҜд»ҘеңЁ$\mathcal{O}(N2^N)$зҡ„жҷӮй–“иӨҮйӣңеәҰе…§иЁҲз®—гҖӮ
зёҪжҷӮй–“иӨҮйӣңеәҰжҳҜ$\mathcal{O}((C+N2^N)\log C)$пјҢе…¶дёӯ$C = 2K\sum A_i$гҖӮиҒҪиӘӘ$C$еҸҜд»ҘеЈ“еҲ°$\mathcal{O}(NK)$дёҚйҒҺжҲ‘дёҚеӨӘжңғOAO

F. Figure

Statement

жңүдёҖеҖӢзҺ©е…·жңү$N$еҖӢйӣ¶д»¶пјҢйӮ„жңү$N-1$еҖӢйҖЈжҺҘйғЁд»¶
з¬¬$i$еҖӢйӣ¶д»¶дёҠйқўжңү$d_i$еҖӢеӯ”
жҜҸеҖӢеӯ”еҸӘиғҪе’ҢдёҖеҖӢйҖЈжҺҘйғЁд»¶йҖЈжҺҘпјҢжҜҸеҖӢйҖЈжҺҘйғЁд»¶еҸҜд»ҘйҖҸйҒҺеӯ”йҖЈжҺҘе…©еҖӢйӣ¶д»¶
е•ҸдҪ жңүеӨҡе°‘дёҚеҗҢзҡ„йҖЈжҺҘж–№ејҸжҠҠжүҖжңүйӣ¶д»¶зө„иЈқеңЁдёҖиө·гҖӮ
жіЁж„Ҹйӣ¶д»¶дёҠйқўзҡ„еӯ”жҳҜзӣёз•°зҡ„пјҢдҪҶжҳҜжүҖжңүйҖЈжҺҘйғЁд»¶йғҪжҳҜзӣёеҗҢзҡ„гҖӮ
д№ҹе°ұжҳҜиӘӘпјҢе…©зЁ®йҖЈжҺҘж–№ејҸ$T_1, T_2$зӣёеҗҢпјҢиӢҘдё”е”ҜиӢҘжүҖжңүеңЁ$T_1$дёӯзҡ„йҖЈжҺҘйғЁд»¶$e_1$пјҢеңЁ$T_2$дёӯйғҪжңүе°ҚжҮүзҡ„$e_2$пјҢе…©з«Ҝзҡ„йӣ¶д»¶з·Ёиҷҹд»ҘеҸҠеӯ”з·Ёиҷҹе®Ңе…ЁзӣёеҗҢгҖӮ

Solution

жҲ‘еҖ‘е°Үжңғд»Ӣзҙ№дёҖзЁ®ж–№жі•иЁҲз®—е®Ңе…Ёең–зҡ„з”ҹжҲҗжЁ№жңүе№ҫзЁ®пјҢдёҰдё”д»ҝз…§и©Іж–№ејҸиЁҲз®—жӯӨе•ҸйЎҢзҡ„зӯ”жЎҲгҖӮ
1. еҒҮиЁӯ$N$еҖӢй»һзҡ„е®Ңе…Ёең–жңү$X$еҖӢз”ҹжҲҗжЁ№гҖӮе°Қж–јжҹҗеҖӢз”ҹжҲҗжЁ№пјҢеҸҜд»Ҙд»»ж„Ҹе®ҡдёҖеҖӢй»һз•¶дҪңж №пјҢдёҰдё”жҠҠ$N-1$еҖӢйӮҠеҠ дёҠз·ЁиҷҹпјҢйҖҷжЁЈзҡ„ж–№ејҸзёҪе…ұжңү$X \times N \times (N-1)!$зЁ®гҖӮ
2. з”ЁеҸҰдёҖзЁ®и§Җй»һдҫҶзңӢйӮҠжңүжЁҷиҷҹзҡ„жңүж №жЁ№ж•ёйҮҸгҖӮжҜҸдёҖжӯҘжҲ‘еҖ‘еҠ дёҠдёҖжўқжңүеҗ‘йӮҠпјҢиҖғж…®еҠ йӮҠеҠ еҲ°зҸҫеңЁзҡ„з”ҹжҲҗжЈ®жһ—пјҢйҖҷдёҖжӯҘеҠ дёҠеҺ»зҡ„жңүеҗ‘йӮҠзҡ„иө·й»һеҸҜд»ҘжҳҜд»»дҪ•дёҖеҖӢй»һпјҢдҪҶжҳҜзөӮй»һеҝ…й ҲжҳҜжҹҗеҖӢйҖЈйҖҡеЎҠзҡ„ж №пјҢиҖҢдё”дёҚиғҪжҳҜеҗҢдёҖеҖӢйҖЈйҖҡеЎҠгҖӮд№ҹе°ұжҳҜиӘӘпјҢеңЁз¬¬$i$ијӘжңү$N \times (N-i)$зЁ®йҒёж“ҮпјҢеӣ зӮәйҷӨдәҶиҮӘе·ұд»ҘеӨ–зҡ„йҖЈйҖҡеЎҠжңү$N-i$еҖӢгҖӮж–јжҳҜзёҪж–№жі•ж•ёжҳҜ$\prod\limits _ {i=1}^{N-1} N (N-i) = N^{N-1} (N-1)!$
3. $X \times N \times (N-1)! = N^{N-1} (N-1)! \Rightarrow X = N^{N-2}$

еҗҢжЁЈең°пјҢжҲ‘еҖ‘е…ҲеҒҮиЁӯжң¬йЎҢзҡ„зӯ”жЎҲжҳҜ$X$пјҢдёҰдё”д»Ө$S = \sum d _ i$
1. иҖғж…®жҹҗзЁ®йҖЈжҺҘйӣ¶д»¶зҡ„ж–№ејҸпјҢдёҰдё”жҠҠ$N-1$еҖӢйӮҠеҠ дёҠз·ЁиҷҹпјҢйҖҷжЁЈзҡ„ж–№ејҸзёҪе…ұжңү$X \times (N-1)!$зЁ®гҖӮ
2. йҰ–е…ҲпјҢе…ҲеңЁжҜҸеҖӢйӣ¶д»¶йҒёж“ҮдёҖеҖӢзү№ж®Ҡеӯ”пјҢеӨ§иҮҙдёҠжҳҜиІ иІ¬з•¶дҪңйӮЈеҖӢй»һе’Ңд»–зҡ„зҲ¶зҜҖй»һйҖЈжҺҘзҡ„еӯ”гҖӮеңЁеүҚ$N-2$жӯҘпјҢжҲ‘еҖ‘жҜҸж¬ЎеҠ дёҠдёҖжўқйӮҠпјҢе…¶дёӯдёҖеҖӢз«Ҝй»һеҸҜд»ҘжҳҜд»»дҪ•йӮ„жІ’з”ЁйҒҺзҡ„еӯ”пјҢдҪҶжҳҜдёҚиғҪжҳҜзү№ж®Ҡеӯ”пјӣиҖҢеҸҰдёҖеҖӢй»һеҝ…й ҲжҳҜжҹҗеҖӢйҖЈйҖҡеЎҠзҡ„ж №зҡ„зү№ж®Ҡеӯ”гҖӮйҖҷжЁЈеңЁз¬¬$i$ијӘжңү$(S - N - i + 1) \cdot (N - i)$зЁ®ж–№жі•гҖӮжңҖеҫҢдёҖжӯҘпјҢеү©дёӢе…©еҖӢйҖЈйҖҡеЎҠпјҢжҲ‘еҖ‘жҠҠд»–еҖ‘зҡ„зү№ж®Ҡеӯ”йҖЈеңЁдёҖиө·гҖӮйҖҷжЁЈжңү$\prod\limits d _ i \cdot \prod\limits _ {i=1}^{N-2} (S - N - i + 1) (N - i)$зЁ®ж–№жі•гҖӮ
3. ж•… $X = \prod\limits d_i \prod\limits _ {i=0}^{N-3} (S - N - i)$

йҖҷйЎҢжҳҜжҲ‘и·‘еҺ»жүҫеҲҘдәәзҡ„и§ЈQQ
жҲ‘зңӢдәҶеҘҪеӨҡи§ЈпјҢеҘҪеғҸйғҪжңүз”ЁеҲ°Cayleyе®ҡзҗҶз”ҡиҮіз”ҹжҲҗеҮҪж•ёд№ӢйЎһзҡ„QAQеҘҪеҷҒеҝғ
жүҫдәҶеҘҪд№…жңҖеҫҢзңӢеҲ°е®ҳж–№youtubeзҡ„и¬ӣи§ЈжүҚиҒҪжҮӮ
